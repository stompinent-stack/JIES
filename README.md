# Jij, ik en Suriname — JIES Website

## Bestanden
- `index.html` — Hoofdpagina
- `style.css` — Stijlen
- `script.js` — Taalwissel & menu
- `images/logo.png` — Jouw logo

## Live zetten via GitHub + Vercel

### Stap 1 — GitHub
1. Ga naar github.com en log in
2. Klik op "New repository"
3. Naam: `jies-website`
4. Klik "Create repository"
5. Upload alle bestanden (drag & drop)

### Stap 2 — Vercel
1. Ga naar vercel.com en log in met GitHub
2. Klik "Add New Project"
3. Kies je `jies-website` repository
4. Klik "Deploy" — klaar!

### Stap 3 — Domeinnaam koppelen
1. Koop `jijikensuriname.nl` via TransIP of Antagonist
2. In Vercel: ga naar je project → Settings → Domains
3. Voeg `jijikensuriname.nl` toe
4. Kopieer de DNS instellingen naar je domeinprovider
5. Binnen 24 uur is je site live op jouw domein

## Aanpassen
- **IBAN**: Zoek `NL XX BANK 0000 0000 00` in index.html en vervang
- **KvK**: Zoek `XXXXXXXX` en vervang
- **Email**: Zoek `info@jijikensuriname.nl` en vervang indien nodig
- **Projecten**: Pas de voortgangsbalken aan (width:72% etc.)

## Blog posts toevoegen
Kopieer een `.blog-kaart` blok in index.html en pas de inhoud aan.
